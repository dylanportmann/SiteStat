# SiteStat

Catalogue local interactif de commandes R pour Stat 1.

## Lancer en local

Sur macOS, double-cliquer sur `Lancer SiteStat.command`.

Sinon:

```bash
npm install
npm run build
npm start
```

Le site ouvre un serveur local et propose un bouton `Arrêter`.

## Développement

```bash
npm install
npm run dev
npm test
npm run build
```
